# typing-speedtest 

A modern and interactive typing speed test application built with HTML, CSS, and JavaScript. The application helps users evaluate and improve their typing skills through timed challenges, real-time performance tracking, and practice exercises.

## Features

- 60-second typing speed challenge
- Easy, Medium, and Hard difficulty levels
- Real-time Words Per Minute (WPM) calculation
- Live accuracy tracking
- Mistake detection and counting
- Instant visual feedback while typing
- Dynamic paragraph generation
- Practice mode with predefined training paragraphs
- Detailed performance summary after each test
- Responsive design for desktop and mobile devices
- Elegant dark-themed user interface

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts
- Font Awesome

## How It Works

1. Select a difficulty level.
2. Click the **Start Typing Test** button.
3. Begin typing the displayed paragraph.
4. The timer automatically starts with the first keystroke.
5. Monitor your speed, accuracy, and mistakes in real time.
6. After 60 seconds, view your final results and performance feedback.
7. Use the practice section to continue improving your typing skills.

## Practice Mode

The application includes a dedicated training section with multiple practice paragraphs covering different topics. Users can select any paragraph to practice without changing the main typing experience.
